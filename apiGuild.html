API 指南

AntPool应用程序编程接口(API)，提供算力、余额等接口，使得用户可以通过自定义编写的软件来访问和控制帐户

请求限制

每10分钟不要超过600次请求，否则我们将禁用您的IP地址

API认证

所有私有API调用都需要进行身份验证。身份验证参数:

  • key - API秘钥
  • nonce - 随机数
  • signature - 签名

API 秘钥

进入“API”配置页面，配置好API访问权限后，单击“生成”按钮，便可生成API访问密钥。

随机数

随机数是一个规则的整数，在您调用接口的过程中，该数字必需不断的增大。 比如：在一次调用中您的nonce配置为1，则在下一次调用中，nonce就最小要配置为2。 通常的做法是将UTC时间戳作为该变量的值。

签名

签名是使用HMAC-SHA256算法对随机数、用户ID、API密钥进行编码后的数据。 签名数据需要使用加密密钥（在生成API密钥时生成）进行生成，并需要转换为十六进制的大写字母或数字。

示例 (Python):

message = userid + api_key + nonce
signature = hmac.new(API_SECRET, msg=message, digestmod=hashlib.sha256).hexdigest().upper()

示例 (Java):

public static String apiSignature(String userId, String key, String nonce, String secret) throws Exception {
    String data = userId+key+nonce;
    Mac hmacSha256 = Mac.getInstance("HmacSHA256");
    SecretKeySpec secretKey = new SecretKeySpec(secret.getBytes(), "HmacSHA256");
    hmacSha256.init(secretKey);
    return Hex.encodeHexString(hmacSha256.doFinal(data.getBytes())).toUpperCase();
}

public static boolean checkApiSignature(String userId, String key, String nonce, String secret, String signature) throws Exception {
    String data = userId+key+nonce;
    Mac hmacSha256 = Mac.getInstance("HmacSHA256");
    SecretKeySpec secretKey = new SecretKeySpec(secret.getBytes(), "HmacSHA256");
    hmacSha256.init(secretKey);
    String signatrueTmp = Hex.encodeHexString(hmacSha256.doFinal(data.getBytes())).toUpperCase();
    if(signature.equals(signatrueTmp)) {
        return true;
    }
    return false;
}

私有方法

矿池统计信息

POST https://antpool.com/api/poolStats.htm

参数:

  • key - API 秘钥
  • nonce - 随机数
  • signature - 签名

返回矿池统计信息，如下所示:

{
"version":"1.0",
"code":0,
"message":"ok",
"data":
  {
  "poolHashrate":"0", //矿池算力（单位为：MH/s）
  "activeWorkerNumber":"0", //有效矿工数
  "poolStatus":"Alive", //矿池状态
  "networkDiff":"0", //当前网络难度
  "estimateTime":"0", //Block期望时间（单位：秒）
  "currentRound":"0", //当前运行时间（单位：秒）
  "totalShareNumber":"0", //总工作量
  "totalBlockNumber":"0" //总Block数
  }
}

矿工算力

POST https://antpool.com/api/workers.htm

参数:

  • key - API 秘钥
  • nonce - 随机数
  • signature - 签名
  • pageEnable - 是否分页返回记录，默认为分页，0:不分页; 1:分页
  • page - 页码，默认为第一页
  • pageSize - 页面大小，默认为每页10条记录

返回用户矿工的算力（单位为：MH/s）、接受数、拒绝数、过期数、重复提交数等，如下所示：

    "prev30m":"0",
    "prev1h":"0",
    "prev1d":"0",
    "accepted":"0",

    "other":"0"

付款记录

POST https://antpool.com/api/paymentHistory.htm

参数:

  • key - API 秘钥
  • nonce - 随机数
  • signature - 签名
  • type - 付款记录获取类型，默认为支付记录， payout: 支付记录; pps: PPS收入记录; pplns: PPLNS收入记录; p2p: P2P收入记录
  • pageEnable - 是否分页返回记录，默认为分页，0:不分页; 1:分页

  • pageSize - 页面大小，默认为每页10条记录
